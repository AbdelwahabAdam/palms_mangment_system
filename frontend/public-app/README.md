# Lifemaker Foundation — Public Palm Discovery

React + TypeScript + Vite + Tailwind CSS public site for discovering sponsored palms.

## Scripts

```bash
npm run dev --workspace=@palms/public-app
npm run build --workspace=@palms/public-app
npm run test --workspace=@palms/public-app
npm run lint --workspace=@palms/public-app
npm run typecheck --workspace=@palms/public-app
```

Local API proxy: `/api` and `/health` → `http://localhost:8000`.
